Теперь о работе с данной программой.

Взаимодействие с программой может происходить либо при помощи jar-файла, либо запуском непосредственно из среды разработки.

При запуске через командную строку в качестве аргументов передаются два числа типа Double - координаты стартовой точки.
Пример : java -jar /path_to_jar 12.0 -12.0

В случае, если одна из координат не будет передана, она будет по умолчанию приравнена 0.0.
Также, в случае, если аргументы не будут переданы в командную строку, обе координаты будут приравнены 0.0.

В программе предоставлены поясняющие комментарии к методам и коду в целом.
